# Stack

A minimal black and white block stacking game. Built as a single HTML file with vanilla JS and Canvas API.

## How to Play

Blocks slide across the screen. Tap, click, or press **Space** to drop the block. Line it up with the one below — any overhang gets sliced off. Miss completely and it's game over.

Land perfectly to keep the full width. Chain perfect landings to grow your block back.

## Run

Open `index.html` in any browser. Works on desktop and mobile.
